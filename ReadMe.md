
#W2云调用库0.1版

##来源
* 不得不说，现在的云服务已经很强大，很多云服务已经完全可以作为微服务替代掉我们所写的繁杂的程序代码。
* 但是，云服务往往又过于复杂，即便我们是浅度的使用。
* 所以，我们在这里提供一个浅度使用的库版本，面向的目标是中小企业和个人所开发的中小项目。
* 顺便说，大家所说的区块链高级概念，在我们这里，也可以理解为云服务，去中心化云嘛。
